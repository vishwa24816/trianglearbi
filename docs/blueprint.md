# **App Name**: ArbiSniffer

## Core Features:

- Real-time Data Stream: Use Binance WebSocket API to receive real-time bid and ask prices for specified trading pairs.
- Arbitrage Path Monitoring: Continuously monitor predefined triangular arbitrage paths (e.g., USDT→BTC→USDC→USDT) for profitable opportunities.
- Profit/Loss Calculation: Calculate potential profit or loss for each arbitrage opportunity based on current bid/ask prices.
- Opportunity Ranking: Present a sortable list of active arbitrage paths, ranked by potential profit or loss.
- Real-time Alerts: Generative AI tool will reason to determine whether it should provide alerts about significant profit opportunities.
- Bid/Ask Display: Display current bid and ask prices for each leg of the arbitrage path.

## Style Guidelines:

- Dark background (#121212) to reduce eye strain and provide a professional look.
- Primary color: Electric lime (#A6FF00) to highlight profitable arbitrage opportunities.
- Accent color: Soft crimson (#FF4961) for indicating potential losses.
- Body and headline font: 'Inter', a sans-serif for readability.
